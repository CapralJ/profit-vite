# Profit Vite

React-приложение для отображения и управления обращениями граждан на карте города.

## Технологии

- React 19
- Vite 7
- Leaflet / React-Leaflet (карты)
- i18next (локализация)
- Lucide React (иконки)

## Установка и запуск

### Требования

- Node.js (версия 18 или выше)
- npm или yarn

### Установка зависимостей

```bash
npm install
```

### Запуск проекта

```bash
npm run dev
```

Приложение будет доступно по адресу: `http://localhost:5173`

### Сборка для продакшена

```bash
npm run build
```

## Структура моковых данных

Файл с моковыми данными расположен по пути:

```
public/data.json
```

### Формат данных

```json
{
  "id": 101,
  "category": "Вывоз мусора",
  "address": "ул. Кабанбай батыра, 45",
  "status": "В работе",
  "created_at": "2025-11-01",
  "description": "Описание проблемы",
  "photo": "https://...",
  "latitude": 51.1282,
  "longitude": 71.4305
}
```
