# Возможности проекта Profit Vite

## Основные функции

### 1. Многоязычность (Локализация)
- Поддержка 3 языков: Русский, Казахский, Английский
- Автоматическое определение языка браузера
- Сохранение выбранного языка в localStorage
- Переключатель языка в шапке приложения
- Полный перевод интерфейса

### 2. Темная тема
- Светлая и темная темы оформления
- Переключатель темы в шапке приложения
- Сохранение выбранной темы в localStorage
- Адаптивная цветовая схема для всех компонентов

### 3. Поиск
- Поиск обращений по категории
- Поиск по адресу
- Поиск в реальном времени (без перезагрузки)
- Регистронезависимый поиск
- Визуальная индикация активного поиска

### 4. Фильтрация
- Фильтрация по статусу обращений:
  - Все обращения
  - В работе
  - Решено
  - Отклонено
- Визуальная индикация активного фильтра
- Комбинирование с поиском

### 5. Сортировка
- Сортировка таблицы по столбцам:
  - ID обращения
  - Дата создания
  - Статус
- Направление сортировки (по возрастанию/убыванию)
- Визуальная индикация активной сортировки

### 6. Пагинация
- Постраничное отображение обращений (10 записей на странице)
- Навигация между страницами
- Отображение текущей страницы и общего количества страниц
- Информация о количестве записей
- Автоматический переход на первую страницу при фильтрации/поиске

### 7. Статистика
- Общее количество обращений
- Количество обращений в работе
- Количество решенных обращений
- Количество отклоненных обращений
- Визуальное отображение статистики в карточках

### 8. Интерактивная карта
- Отображение обращений на карте города (Leaflet)
- Маркеры с координатами каждого обращения
- Кликабельные маркеры для просмотра деталей
- Автоматическое центрирование карты
- Группировка маркеров при приближении/отдалении

### 9. Модальное окно
- Детальная информация об обращении:
  - ID обращения
  - Категория проблемы
  - Адрес
  - Статус с цветовым индикатором
  - Дата создания
  - Подробное описание
  - Фотография проблемы (если есть)
  - Координаты на карте
- Закрытие по клику вне окна или на кнопку закрытия

### 10. Таблица обращений
- Отображение списка обращений в табличном виде
- Столбцы:
  - ID
  - Категория
  - Адрес
  - Статус (с цветовым бейджем)
  - Дата создания
- Кликабельные строки для просмотра деталей
- Адаптивный дизайн таблицы

### 11. Бейджи статусов
- Цветовая индикация статусов:
  - В работе (синий)
  - Решено (зеленый)
  - Отклонено (красный)
- Автоматический перевод статусов на выбранный язык

### 12. Адаптивный дизайн
- Корректное отображение на различных устройствах
- Адаптация интерфейса для мобильных устройств
- Гибкая сетка макета

## Технические особенности

- React 19 с использованием хуков
- Vite 7 для быстрой разработки и сборки
- i18next для локализации
- React-Leaflet для работы с картами
- Lucide React для иконок
- Мемоизация для оптимизации производительности
- Загрузка данных из JSON файла
- Обработка состояний загрузки и ошибок
